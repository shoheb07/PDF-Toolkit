# PDF-Toolkit
This project is useful for students, office workers, and professionals who frequently work with PDF documents.

PDF Toolkit

Overview

PDF Toolkit is a Python desktop application that provides multiple PDF utilities such as merging, splitting, and page extraction through a simple graphical interface.

The application helps users manage PDF files efficiently without requiring third-party online tools.

---

Features

- Merge multiple PDFs
- Split PDF into individual pages
- Extract specific pages
- Simple GUI
- Lightweight application

---

Technologies Used

- Python
- Tkinter
- PyPDF2

---

Installation

Install required library:

pip install PyPDF2

---

Run Project

python pdf_toolkit.py
